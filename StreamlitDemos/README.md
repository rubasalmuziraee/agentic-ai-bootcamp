
# 🚀 Streamlit Lab Project

This project is a complete hands-on implementation of Streamlit fundamentals, including input components, selection tools, data display, and interactive applications.

---

## 📌 Features

### 1. Basic Input Components
- Text input
- Password input
- Number input
- Button interaction

### 2. Selection Components
- Slider
- Time range slider
- Checkbox
- Radio buttons
- Select box

### 3. Data Display
- Lists and dictionaries
- DataFrames (tables)
- JSON display
- File uploader

### 4. Interactive Features
- Session state counter
- Contact form using `st.form`
